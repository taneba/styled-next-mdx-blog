# styled-next-mdx-blog

## Technology stack
- React
- Next.js
- styled-components
- mdx

## How to use
Clone repo:

```
git clone https://github.com/taneba/styled-next-mdx-blog.git
```

Install the dependencies
```
yarn install
```

setup local server
```
yarn dev
```
